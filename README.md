# spotify-clone-frontend

A responsive Spotify-like UI built with HTML and CSS to practice frontend layouting and UI components.

## Live Demo
This project is deployed using GitHub Pages: `https://himanth-dev.github.io/spotify-clone-frontend`.

## Features
- Responsive layout with sidebar, playlist area, and player controls  
- Reusable UI components and CSS structure (flexbox/grid)  
- Basic interactive player controls (UI only)  
- Static demo data (no backend)

## Tech Stack
- HTML5, CSS3

## How to run locally
1. Download or clone the repository.  
2. Open `index.html` in your browser.

## Folder Structure
```
project/
├── index.html
├── style.css
└── assets/
    └── images and icons used in UI
```

## Notes
This is a front-end clone intended for UI practice. No authentication or music streaming backend is implemented.
